• Strings are useful for holding data that can be represented in text form.
  • String is a primitive data type in JavaScript. A string is textual content.
  • Since, string is character index, it can be accessed using for loop .
  • Strings are very useful when communicating information from the program to the user of the program.
  • The technical description of a String is: an array of characters. The informal view of a string is a sentence.
